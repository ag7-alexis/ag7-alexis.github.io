AG7-Chat

Application web inspiré fortement de Messenger permettant d'envoyer des messages à d'autres utilisateurs. Les utilisateurs peuvent aussi ajouter des contacts et modifier leur photo de profil.

L'objectif de cette application pour la partie Back-End été de tester la programmation réactive (mis en place seulement sur l'envoi de message pour l'instant) ainsi que la configuration d'un websocket. Et pour la partie Front-End de mettre en place du Lazy Loading.

Front-End

Le Front-End est fait avec Angular 10 avec la précieuse aide de Halfmoon et d'AngularMaterial. J'utilise Stomp et SocketJS pour le websocket.

Back-End

La partie Back-End est réalisée en Java 11 avec le Framework Spring. J'utilise ici SpringBoot, Spring Security ainsi que Spring Webflux.

Base de données

J'utilise une base de données NoSQL ave MongoDB , la persistance des données est assuré à l'aide de JPA et Hibernate.

AG7-Project

Application web qui permet de gérer des projets selon la méthode Kanban. Les utilisateurs peuvent créer des projets, les partager à d'autres utilisateurs et gérer les tâches a réaliser pour ce projet.

L'objectif de cette application été découvrir Angular un peu plus en profondeur pour ce qu'y est de la partie Front-End. Et pour la partie Back-End j'ai appris à utiliser Spring Security pour sécuriser les requêtes avec des JWT.

Le Front-End est fait avec Angular 8 avec la précieuse aide de Bootstrap et d'AngularMaterial.

AG7-JobHunt

Application web qui permet de gérer des offres d'emploi pour les développeurs. Les utilisateurs peuvent avoir différents rôles qui leur attribueront des permissions spécifique, par exemple un manager pourra ajouter des offres d'emploi lisible pour les utilisateurs inscrits.

J'ai fait cette application pour essayer la gestion des rôles avec Spring Security et par la même occasion, j'ai pu travailler avec une API extérieure.

Application

Cette application est réalisée en Java 11 avec le FrameWork Spring dont j'utilise principalemet Spring MVC et Spring Security. Les vues sont faites avec Thymeleaf et Bootstrap. Et j'ai ajouté l'API de recherche d'emploi de github pour avoir du contenu.

Github de l'application

Base de données

J'utilise une base de données PostgreSQL, la persistance des données est assuré à l'aide de JPA et Hibernate.

AG7-GeoAPI

API publique permettant d'obtenir des données sous le format JSON, tel que la liste des régions de France ou encore la liste des communes d'un département par exemple. Les utilisateurs qui souhaitent utiliser l'API dans leurs projets auront juste à s'inscrire pour obtenir une clé d'API.

L'objectif de cette API été de pouvoir garder un temps de réponse de l'API relativement cours sur une base de données beaucoup plus rempli comparé a celle que j'ai l'habitude d'utiliser (+ 5000 enregistrements).

Documentation

La documentation est faite avec Angular 8 avec la précieuse aide de Bootstrap et d'AngularMaterial. Des zones de tests sont disponibles pour essayer des requêtes vers l'API.
