AG7-Chat

Application web inspiré fortement de Messenger permettant d'envoyer des messages à d'autres utilisateurs. Les utilisateurs peuvent aussi ajouter des contacts et modifier leur photo de profil.

L'objectif de cette application pour la partie Back-End été de tester la programmation réactive (mis en place seulement sur l'envoi de message pour l'instant) ainsi que la configuration d'un websocket. Et pour la partie Front-End de mettre en place du Lazy Loading.

Front-End

Le Front-End est fait avec Angular 10 avec la précieuse aide de Halfmoon et d'AngularMaterial. J'utilise Stomp et SocketJS pour le websocket.

Back-End

La partie Back-End est réalisée en Java 11 avec le Framework Spring. J'utilise ici SpringBoot, Spring Security ainsi que Spring Webflux.

Base de données

J'utilise une base de données NoSQL ave MongoDB , la persistance des données est assuré à l'aide de JPA et Hibernate.
