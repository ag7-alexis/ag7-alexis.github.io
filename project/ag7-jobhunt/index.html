AG7-JobHunt

Accéder au projeten cliquant ici !Accéder au code du projeten cliquant ici !

Cette application web permet de trouver des offres d’emploi pour des postes de développeur un peu partout dans le monde. Les utilisateurs peuvent avoir différents rôles qui leur permettront par exemple pour un manager de proposer une offre d’emploi de développeur web aux utilisateurs lambda inscrit.

Technologies utilisées

Pour la réalisation de cette application, j’ai utilisé Java, Spring et Thymelead. J’utilise plus particulièrement SpringMVC et SpringSecurity. Pour récupérer des offres d’emploi, je passe par l’API externe de Github qui propose une multitude d’offres d’emploi. J’utilise une base de données PostgreSQL, la persistance des données est assuré à l’aide de JPA et Hibernate.

La création de cette application m’a permis de travailler avec une API externe. J’ai aussi pu travailler sur la gestion des rôles et permissions des utilisateurs grâce à SpringSecurity, en effet un manager n’aura pas les mêmes droits qu’un administrateur ou qu’un simple utilisateur.
