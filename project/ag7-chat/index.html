AG7-Chat

Accéder au projeten cliquant ici !

Cette application web est une application de messagerie fortement inspirée de Messenger. Les utilisateurs peuvent ajouter des contacts, envoyer des messages en direct à leurs contacts et changer leurs photos de profils.

Technologies utilisées

Ce projet a été réalisé en Angular pour la partie front-end. Pour la partie back-end, j’utilise Java et Spring. Pour Spring, je suis partie sur WebFlux et un WebSocket pour réaliser l’échange de messages entre deux clients en direct. Pour ma base de données, je suis partie sur une base de données NoSQL en MongoDB.

Grâce à cette application, web j’ai pu travailler pour la première fois avec un WebSocket et j’ai pu enfin tester l’utilisation d’une base de données NoSQL. J’ai découvert aussi l’optimisation et l’encodage d’image directement en base de données. Ce projet m’a donc permis de continuer à monter en compétences sur Java, Spring et Angular.
