Portfolio

Mon portfolio de développeur web permet de mettre en avant tous les projets sur lesquels j’ai pu travailler ou encore les technologies que j’aime utiliser. Vous pourrez retrouver pour chaque projet le lien pour y accéder, le lien pour le github de certains ainsi que différents screens et explications sur les technologies utilisées ou encore le contexte de la réalisation.

Technologies utilisées

J’ai réalisé mon portfolio de développeur web selon la JAMStack. Pour la partie back-end, j’utilise le CMS headless Strapi qui me permet de créer facilement une API en Javascript sous NodeJS. Mon front-end est basé sur VueJS, j’utilise GridSome pour pouvoir transformer mon application web en VueJS à un site statique. Je fais communiquer mon front-end avec mon API grâce à GraphQL

La réalisation de mon portfolio m’a permis de découvrir la JAMStack qui je pense est bon à connaître en tant que développeur web surtout en ce moment, car c’est actuellement très rependus. J’ai pu aussi pour la première fois travailler avec Strapi et GridSome qui je trouve font un bon combo pour la réalisation de site statique sous VueJS.
