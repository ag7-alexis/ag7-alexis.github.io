Localio

Accéder au projeten cliquant ici !

Ce projet a été réalisé dans le cadre de ma licence professionnelle MIW en groupes de quatre. L’objectif était de créer une application web sur laquelle des commerçants ont la possibilité d’exposés leurs commerces sur notre carte et des utilisateurs voulant supporté les commerçants au tour de chez eux puissent se renseigner sur les commerces. L’application propose donc une page sur laquelle n’importe quel utilisateur puisse faire des recherches de commerce sur une carte. Pour que les commerçants aient un aperçu de l’intérêt de notre application, des statistiques leurs sont proposés ainsi qu’un espace commentaire afin d’avoir l’avis de leurs clients.

Technologies utilisées

Pour la réalisation de ce projet, nous avions une seule chose d’imposer qui était le fait d’utiliser le framework Laravel. Nous avons fait le choix d’utiliser différentes librairies telles que Leaflet pour la carte, ChartJS pour la réalisation des différents diagrammes pour les statistiques, VueJS ou encore Halfmoon pour le design de la partie back-office. Nous utilisons aussi des API externes telles que l’API du gouvernement français qui nous permet de localiser des adresses par exemple ainsi que l’API d’authentification de Google pour proposer aux utilisateurs de se connecter via le compte Google.

Ce projet a été pensé, designé et développé en l’espace de seulement deux mois. Pour mener à bien la réalisation de ce projet dans un laps de temps aussi réduit nous avons du faire preuve une très grande organisation. Pour nous organiser, nous avons utilisé la méthode Kanban afin de bien nous répartir les tâches et ne pas être trop dépendant les uns des autres, et nous avons utiliser git/github pour le versionning.
