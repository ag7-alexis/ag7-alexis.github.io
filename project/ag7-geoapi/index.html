AG7-GeoAPI

Accéder au projeten cliquant ici !Accéder au code du projeten cliquant ici !

Cette API permet de récupérer des données géographiques Français sous le format JSON. Elle propose une multitude de end-points offrant la possibilité de récupérer la liste des communes d’un département ou encore la liste de toutes les régions par exemple. Les utilisateurs peuvent tester l’API directement dans la partie documentation ou obtenir une clé d’API pour l’utiliser dans leur projet. La partie test de l’API permet de générer des URL et de voir ce que retourne l’API pour cette requête, pour construire sa requête l’utilisateur peut choisir les données qu’il veut ou non que l’API lui renvoi.

Technologies utilisées

Cette API a été réalisée en Java avec l’utilisation des frameworks Spring et SpringSecurity. L’API rempli les critères pour être qualifiée d’API REST, elle ne remplit juste pas le critère facultatif selon lequel elle doit pouvoir envoyer du code à la demande. La partie documentation a été développée en Angular. J’utilise une base de données PostgreSQL, la persistance des données est assuré à l’aide de JPA et Hibernate.

Ce projet est un projet personnel qui m’a permis de continuer à monter en compétences sur Angular et Java. L’objectif principal de ce projet été de créer une API facile d’utilisation et performante. J’ai dû faire attention au temps de réponse de mes différents end-points car la France possède un nombre important de communes il fallait donc optimiser la structure de la base de données ainsi que mes algorithmes de recherche de données pour garantir un temps de réponse le plus court possible.
